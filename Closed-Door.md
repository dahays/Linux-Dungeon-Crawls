
# The Closed Door

## Overview
An introductory Linux Dungeon Crawl.  Students must navigate the dungeon using basic file navigation, creation, and manipulation commands.  These Linux Dungeon Crawls will build upon skills of the previous Crawls in the series.  Look to the strange_manuscript for clues.

**Skills Practiced**
- File creation (`touch`)
- File editing (`vim`, `nano`)
- File and directory management (`cd`, `cat`, `mv`, `mkdir`, `rmdir`, `pwd`, `ls`, `tree`, `cp`, `rm` )

---

## Setup Instructions

Open a terminal.

Download the setup script:
```bash
wget -O setup_intro.sh https://raw.githubusercontent.com/dahays/Linux-Dungeon-Crawls/main/setup_intro.sh
```
Make the script executable.
```bash
chmod +x setup_intro.sh
```
Run the script.
```bash
./setup_intro.sh
```
Enter the lair.
```bash
cd ~/dungeon_intro
```
Confirm success.
```bash
./check_intro.sh
```
