# 🏰 The Archivist’s Vault 🔮

(Linux Dungeon Crawl Intro 1B)

## Overview
You stand before the Archivist’s Vault, a chamber shrouded in mystery. Its secrets will not yield to force, only to careful craft and understanding.

This dungeon introduces basic Linux scripting, directory creation, and file management. You will forge a vault, create sacred artifacts, and use a ritual script to satisfy the Guardian’s watchful eye.

The strange_manuscript.txt contains your instructions. Read it carefully — it will guide your actions without revealing exact commands.

## Setup Instructions

Open a terminal.

Download the setup script:
```bash
wget -O setup_vault.sh https://raw.githubusercontent.com/dahays/Linux-Dungeon-Crawls/main/setup_vault.sh
```
Make the script executable:
```bash
chmod +x setup_vault.sh
```
Run the setup script:
```bash
./setup_vault.sh
```
Enter the dungeon:
```bash
cd ~/dungeon_vault/vault_entrance
```
Confirm success:
```bash
./check_vault.sh
```
